# Liveline

Liveline is a desktop ticker app that scrolls headlines or updates from user-selected websites across your screen.

## Features
- Pulls data from RSS feeds or websites
- Scrolls text left-to-right or right-to-left
- Customizable speed, font, and colors (coming soon)

## Installation
```bash
git clone https://github.com/yourusername/liveline.git
cd liveline
pip install -r requirements.txt
python ticker.py
